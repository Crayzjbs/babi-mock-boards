# Architecture Board Exam Reviewer

An Anki-style flashcard quiz web app for architecture board exam preparation. Built with Next.js 14, TypeScript, Tailwind CSS, shadcn/ui, and Supabase.

## Features

- **Topic-based Study Sessions** - Study by specific topics or all topics combined
- **Flip-to-Reveal Flashcards** - Smooth 3D flip animation to reveal answers
- **Self-Rating System** - Rate your knowledge with Again, Hard, Good, Easy (Anki-style)
- **Shuffle Mode** - Randomize questions per topic or globally
- **Session Progress Tracking** - Track reviewed cards, known vs needs review
- **Review Missed Cards** - Focus on cards you marked as Again/Hard
- **Keyboard Shortcuts** - Space/Enter to flip, 1-4 for ratings
- **Light/Dark Mode** - Clean, minimal design with full theme support
- **Responsive Design** - Works seamlessly on mobile and desktop

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: shadcn/ui
- **Database**: Supabase (PostgreSQL)
- **Icons**: Lucide React
- **Notifications**: Sonner

## Setup Instructions

### 1. Prerequisites

- Node.js 20+ installed
- A Supabase account and project ([supabase.com](https://supabase.com))

### 2. Environment Variables

Create a `.env.local` file in the root directory:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Get these values from your Supabase project: **Project Settings > API**

### 3. Database Setup

1. Go to your Supabase project dashboard
2. Navigate to **SQL Editor**
3. Run the contents of `supabase_schema.sql` to create the `cards` table

### 4. Import Flashcard Data

1. Prepare your CSV file with columns: `topic`, `question`, `answer`
2. In Supabase dashboard: **Table Editor > cards table**
3. Click **Insert > Import data from CSV**
4. Upload your CSV (see `flashcard_data_sample.csv` for format)

### 5. Install Dependencies & Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## CSV Data Format

Your `flashcard_data.csv` should have this structure:

```csv
topic,question,answer
History of Architecture,Who designed the Parthenon?,Ictinus and Callicrates
Structural,What is the modulus of elasticity of steel?,29000 ksi or 200 GPa
Standards & Codes,What is the minimum ceiling height?,2.40 meters
```

The `id` and `created_at` columns are auto-generated by Supabase.

## Keyboard Shortcuts

- **Space / Enter** - Flip card to reveal answer
- **1** - Rate as "Again" (needs review)
- **2** - Rate as "Hard"
- **3** - Rate as "Good"
- **4** - Rate as "Easy"

## Design Philosophy

- **Minimal & Clean** - No gradients, no heavy shadows, no decorative clutter
- **Whitespace & Typography** - Clear hierarchy, generous padding
- **Subtle Borders** - 1px borders over color-heavy UI
- **Purposeful Motion** - Smooth flip animation as the signature interaction
- **Neutral Palette** - Whites, grays, blacks with restrained accent color
